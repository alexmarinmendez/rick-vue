# rick-vue
## Description
🎯 A demo repo for my classes at CoderHouse. A really simple VueJs app accessing the Rick and Morty API.
* 👀 Based in the Udemy Course "Vue 3 - Composition API, Vuex, API Rest - Rick And Morty" from Carlos Cordova
